* Jetpack
* All in One SEO Pack
* UpdraftPlus
* Broken Link Checker
* iThemes Security
* Smush
... more comming

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'No Premium'
3. Activate No Premium from your Plugins page.

= From WordPress.org =

1. Download No Premium.
2. Upload the 'no-premium' directory to your '/wp-content/plugins/' directory.
3. Activate No Premium from your Plugins page.
